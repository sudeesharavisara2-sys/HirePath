# HirePathAI – Phase 1

This package completes the naming and solution-structure conversion from `ResumeWeb` to `HirePathAI`.

## Open and run
1. Open `HirePathAI.sln` in Visual Studio 2022.
2. Right-click `HirePathAI.Web` and choose **Set as Startup Project**.
3. Restore NuGet packages.
4. Build the solution (`Ctrl+Shift+B`).
5. Run with `Ctrl+F5` or `F5`.

`ResumeTrainer` remains unchanged because it is the ML model-training utility.
